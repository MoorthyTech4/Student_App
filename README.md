# Favourite Student List App

## Overview

Favourite Student List is a React JS application that allows users to view a list of students, add students to a favourites list, and manage favourite students using React Context API. The application demonstrates React Router for navigation, useContext for global state management, and dynamic rendering of student data.

## Features

* View a list of students.
* Add students to the favourites list.
* Remove students from the favourites list.
* Navigate between Student List and Favourite Students pages using React Router.
* Global state management using Context API.
* Dynamically add new students through input fields.
* Responsive and clean user interface built with Tailwind CSS.

## Technologies Used

* React JS
* React Router DOM
* Context API
* Tailwind CSS
* JavaScript (ES6+)

## Project Structure

* **Student List Page** – Displays all available students.
* **Favourite Students Page** – Displays students added to favourites.
* **Student Context** – Manages favourite student data globally.
* **Student Card Component** – Reusable component for displaying student information.

## Learning Objectives

This project demonstrates:

* React Functional Components
* React Hooks (`useState`, `useContext`)
* Context API for state management
* React Router for page navigation
* Conditional Rendering
* Component Reusability
* Event Handling and Form Inputs

## How to Run

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the application in your browser.
